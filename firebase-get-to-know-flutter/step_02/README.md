# 🚀 Proyecto Codelab de Firebase y Flutter 🚀

¡Bienvenido! Este proyecto es una implementación del codelab oficial "Descubre Firebase para Flutter". Es una excelente plantilla para cualquiera que quiera aprender a integrar una aplicación de Flutter con los servicios de Firebase.

**¡Anímate a hacer un `fork` de este repositorio y a experimentar por tu cuenta!**

---

## 🌟 Características Principales

*   🔐 **Autenticación de Usuarios:** Registro e inicio de sesión con Firebase Authentication.
*   ☁️ **Base de Datos en Tiempo Real:** Sincronización de datos al instante con Cloud Firestore.
*   📱 **UI Moderna con Flutter:** Interfaz limpia y adaptable construida con Flutter.

## 🏃‍♂️ Cómo Empezar

1.  **Haz un `fork` o clona este repositorio.**
2.  **Navega al directorio del proyecto:**
    ```bash
    cd codelabs/firebase-get-to-know-flutter/step_02
    ```
3.  **Instala las dependencias:**
    ```bash
    flutter pub get
    ```
4.  **¡Ejecuta la aplicación!**
    ```bash
    flutter run
    ```
